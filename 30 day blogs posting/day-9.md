Percentage calculations are fundamental to business, finance, and everyday life. From calculating profit margins to determining discounts, mastering these calculations saves time and prevents costly errors.

### Essential Percentage Formulas

#### Basic Percentage
```
Percentage = (Part / Whole) × 100
Example: What is 25% of 80?
= (25 / 100) × 80 = 20
```

#### Finding Percentage of a Number
```
Result = Number × (Percentage / 100)
Example: 15% of 250 = 250 × 0.15 = 37.5
```

#### Percentage Change
```
Change = ((New - Old) / Old) × 100
Example: Price changed from $50 to $65
= ((65 - 50) / 50) × 100 = 30%
```

#### Reverse Percentage
```
Original = Final / (1 + Percentage/100)
Example: After 20% tax, total is $120
= 120 / 1.20 = $100 original
```

### Common Business Applications

1. **Profit Margin**: (Revenue - Cost) / Revenue × 100
2. **Markup**: (Selling Price - Cost) / Cost × 100
3. **Discount**: Original Price × (Discount% / 100)
4. **Tax Calculation**: Subtotal × Tax Rate

### Quick Percentage Tips

- 10% = Move decimal one place left
- 25% = Divide by 4
- 50% = Divide by 2
- 33% ≈ Divide by 3
