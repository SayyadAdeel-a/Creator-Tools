While meta descriptions don't directly impact rankings, they significantly influence click-through rates. A compelling description can mean the difference between a click and a bounce.

### Meta Description Best Practices

#### Length
- Optimal: 150-160 characters
- Maximum: ~920 pixels
- Google often shows 155-160 characters

#### Structure
- Include primary keyword
- Add a clear CTA
- Highlight unique value

#### Writing Style
- Active voice
- Benefit-focused
- Natural language (not keyword stuffed)

### Meta Description Formula

```
[Hook/Problem] + [Solution] + [CTA]
Example:
Struggling with weak passwords? Use our free password generator to create unbreakable passwords in seconds. Try it now!
```

### Description vs Title

- **Title**: What users click on
- **Description**: What convinces them to click
- Both work together for CTR

### Common Mistakes

- Duplicate descriptions
- Missing descriptions
- Too long or too short
- No call-to-action
- Keyword stuffing
