Converting numbers to words is essential for legal documents, financial transactions, checks, and contracts. This prevents fraud and ensures clarity in important transactions.

### When to Use Number to Words

1. **Legal Documents** - Contracts, agreements
2. **Financial Instruments** - Checks, money orders
3. **Invoicing** - Total amounts in words
4. **Real Estate** - Property values in contracts

### Number Words Examples

| Number | Written |
|--------|---------|
| 1 | One |
| 25 | Twenty-five |
| 100 | One Hundred |
| 1,000 | One Thousand |
| 1,000,000 | One Million |

### Currency Formats

Standard check writing format:
- $125.50 = One Hundred Twenty-Five and 50/100 Dollars
