While HEX codes dominate web development, most design tools output RGB values. Learning to quickly convert RGB to HEX is a valuable skill that speeds up your design workflow significantly.

### Manual RGB to HEX Conversion

Converting RGB to HEX involves converting each decimal value to its hexadecimal equivalent:

1. **Red (0-255)** → Two HEX digits
2. **Green (0-255)** → Two HEX digits
3. **Blue (0-255)** → Two HEX digits

Example: rgb(255, 174, 55)
- 255 = FF
- 174 = AE
- 55 = 37
- Result: #FFAE37

### Using RGB Values in CSS

Modern CSS supports both HEX and RGB:
```css
/* Both produce the same color */
background-color: #FFAE37;
background-color: rgb(255, 174, 55);
```

### When to Use RGB vs HEX

**Use RGB when:**
- Working with color opacity (rgba)
- Converting from design software
- Using CSS custom properties dynamically

**Use HEX when:**
- Writing static CSS
- Sharing colors with developers
- Working in HTML/CSS environments
